# LANG=ru_RU.UTF-8

Установить:
	kdebase-workspace-python-applet
	python-crypto

Путь к логу ошибок сессии:
	/dev/shm/errorMailChecker[EPOH_time].log

Путь к логу возможного отладочного вывода сессии:
	/dev/shm/outMailChecker[EPOH_time].log

Путь к логу апплета:
	~/mailChecker.log
	или:
	~/Документы/mailChecker.log

Путь к файлу конфигурации апплета:
	~/.config/mailChecker/mailChecker.conf

Для запуска апплета кликнике иконку "письмо".
Если ключ шифрования паролей не существует,
то его необходимо создать.
